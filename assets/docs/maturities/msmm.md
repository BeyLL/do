# 微服务成熟度模型（MSMM）

基于 《生产微服务》

```maturity
 - 微服务成熟度模型
   - 稳定性和可靠性
     - 开发周期
     - 部署管道
     - 服务依赖
     - 路由和服务发现
     - 服务和端点的解除
   - 伸缩性和高性能
     - 增长规模
     - 资源的有效利用
     - 资源感知
     - 容量规划
     - 依赖项的伸缩
     - 流量管理
     - 任务处理
     - 可伸缩的数据存储
   - 容错和灾备
     - 避免故障点
     - 故障场景
     - 弹性测试
     - 故障检测和修复
   - 监控
     - 关键性的度量指标
     - 日志
     - 仪表盘
     - 告警
     - 轮班待命
   - 文档化和理解
     - 微服务文档
     - 微服务理解
```
